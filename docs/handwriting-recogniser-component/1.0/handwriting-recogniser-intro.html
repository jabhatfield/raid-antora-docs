Introduction

Handwriting Recogniser implements DJL to train a neural network on images of handwritten numbers within the range of 0-9. It can then generate the probabilities of an input image matching each of these numbers.

Get started

See one or more of the following ways to get started, depending on preference of learning approach:

  • Perform the Install and run quickstart to run Handwriting Recogniser.

  • Complete the Handwriting Recogniser tutorial to gain practical experience of the API.

  • Read the Handwriting Recogniser deep dive to gain a deeper theoretical understanding of the chatbot.

  • See the API specification for fine details of the API.
